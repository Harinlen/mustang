Execute one or more tool calls in a single batch. When REPL mode is active, primitive tools (Bash, FileRead, FileEdit, FileWrite, Glob, Grep, Agent, etc.) are only accessible through this tool.

Submit an array of tool calls. Each call specifies the tool name and its input parameters. Results are returned per-call in indexed <repl_result> blocks.

Read-only tools in the batch may run concurrently; mutating tools run serially in order.
